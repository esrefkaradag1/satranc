Bu klasör: resmi Stockfish kaynak kodu ve macOS için önceden derlenmiş ikili dosya (stockfish-macos-m1-apple-silicon).
Tarayıcı analizi için doğrudan bu ikili kullanılmaz — tarayıcıda Stockfish 18 WASM gerekir.

Projede tarafımızdan kullanılan sürüm: npm paketi "stockfish" (Chess.com / nmrugg stockfish.js, SF18 lite-single).
Kopyalama: npm install sonrası veya "npm run sync:stockfish" ile public/stockfish dizinine yazılır.

Kaynak upstream: https://github.com/official-stockfish/Stockfish
